# SELENIUM-TESTING---OPENING-CHROME-AUTOMATED
A simple Selenium WebDriver project in Java that automates opening Google Chrome. Ideal for beginners to learn browser automation, driver setup, and the basics of running automated web tests.
This project demonstrates the automation of opening Google Chrome using Selenium WebDriver in Java. It serves as a simple introduction to browser automation, showcasing how Selenium interacts with web browsers programmatically.
